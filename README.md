# 🧬 **PROMPT EVOLVER**
*Self-evolving prompt optimizer using genetic algorithms.*

---
### **🔥 What It Does**
Feed it weak, vague, or underperforming prompts.
It **mutates, cross-breeds, and evolves** them into **god-tier outputs** using genetic algorithms.

---
### **🛠️ Tech Stack**
- **Language:** Python
- **Core:** Genetic Algorithms, LLM Fine-Tuning
- **Optimization:** Fitness-based selection, crossover, mutation
- **Use Case:** AI prompt engineering, automation, performance tuning

---
### **🚀 Quick Start**
1. **Clone:**
   ```bash
   git clone https://github.com/svanaprins/prompt-evolver.git
   cd prompt-evolver
   ```
2. **Install:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run:**
   ```bash
   python evolver.py --input "your weak prompt here"
   ```

---
### **🎯 Features**
- **Automated Evolution:** No manual tuning—just feed it prompts.
- **Customizable:** Adjust mutation rates, population size, and fitness functions.
- **Scalable:** Works with any LLM (GPT, Claude, Mistral, etc.).
- **Open-Source:** MIT License. Fork, modify, contribute.

**Input:**

### **🔧 Configuration**
Edit `config.yaml` to tweak:
- Population size
- Mutation rate
- Fitness metrics (e.g., engagement, clarity, originality)
